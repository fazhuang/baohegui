# 包合规 - 产品文档

## 产品概述

- 产品名称：包合规
- 专业名称：华招国际标准化文书合规智能审核系统
- 目标用户：招标代理机构、政府采购业务部门
- 使用场景：招标文件在公共资源交易平台正式发布前，进行内部合规自检

## 行业背景

- 2026年底起，所有政府采购平台强制接入AI辅助审查
- 公共资源交易平台的AI审查为辅助性质，不承担法律风险
- 质疑投诉由财政部门处理，招标文件编制方承担最终法律责任
- 包合规定位：发布前自检 + 合规风险全面兜底

## 价值主张

1. **防拦截**：对齐公共资源交易平台审查规则，降低驳回率
2. **防质疑**：AI识别潜在质疑点，提前规避争议条款
3. **防处罚**：以《政府采购法》为准绳，降低行政处罚风险
4. **可追溯**：完整合规记录，可应对质疑答复和审计要求
5. **低成本**：替代大部分人工审查，降低法务成本

|------|------|------|----------|

## 核心指标

- 单份文件审查时间 ≤ 3分钟
- 规则引擎准确率 ≥ 95%
- 大模型语义审查召回率 ≥ 85%
- 平台拦截预测准确率 ≥ 90%
