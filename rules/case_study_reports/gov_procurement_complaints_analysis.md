### 新增发现的违规模式

#### P12: 指定特定检测机构出具报告
**来源**: 四川通报（摄像机设备采购项目）
**描述**: 采购文件要求提供某单位所属检测机构出具的检测报告才能得分，但事实上具有法定检测资质的机构均可出具有效报告
**检测**: 检测评分标准中是否限定特定检测机构出具报告
**新增规则**: forbidden_pattern 增强 FORB-L04

#### P13: 供应商变造/篡改证明材料
**来源**: 四川通报（机械设备采购项目）、成都（虚假检测报告案）
**描述**: 供应商在投标文件中修改检测报告的技术参数，或使用虚假检测报告谋取中标。这是财政部通报最高频的违规类型之一
**检测**: 结合社会信用代码验证检测报告编号和参数是否与官方备案一致
**新增规则**: AI-SME-DECLARE + 检测报告验证

#### P14: 评审专家未按标准扣分
**来源**: 四川通报（教学实训中心项目）、海南大学观澜湖项目
**描述**: 供应商对技术参数负偏离响应，评审专家未按采购文件规定扣分
**检测**: 对比负偏离响应与评分结果
**新增规则**: F004

#### P15: 恶意串通（MAC地址/IP一致）
**来源**: 四川通报（出版服务项目）
**描述**: 多家供应商上传投标文件使用相同MAC地址、IP地址、CPU代码，存在借用资质串通投标
**检测**: 供应商投标文件的电子特征信息比对
**新增规则**: AI-BID-RIGGING

#### P16: 代理机构超标准收费
**来源**: 四川通报（家具采购项目）
**描述**: 代理机构实际收取服务费比标准多收近8000元
**检测**: 代理服务费计算对照标准费率
**新增规则**: C类规则扩展

#### P17: 符合性审查程序错误
**来源**: 甘肃食品安全监测项目
**描述**: 评标委员会将非实质性条款缺失（产地信息未填写）认定为符合性审查不通过
**检测**: 检查评标标准中实质性条款标注与非实质性条款的区分
**新增规则**: D类规则扩展

#### P18: 虚假检测报告+产品注册证不符
**来源**: 福建彩超项目（投诉方主张）、四川成都（虚假检测报告）、内江（虚假响应）
**描述**: 供应商声称产品具有特定注册证或检测报告，但与药监局官网信息不符
**检测**: 跨库验证（药监局数据库 vs 投标文件注册证信息）
**新增规则**: AI-REG-VERIFY

### 新发现的模式（需新增/增强）✗

#### P1: 混包限制竞争（Improper Bundling）
**来源**: 宁夏2026-8号（智慧烹饪项目）
**描述**: 将完全不同性质的采购标的（厨卫用具+计算机软件+工程改造）捆绑在一个标段，限制了具备单一品类优势的供应商参与。
**检测**: 检测同一标段内 project_categories 跨越非关联品类

#### P2: 废止/不存在标准引用
**来源**: 青海职业技术大学项目
**描述**: 引用已废止的国家标准（如GB18584-2001）甚至不存在的标准（如GB2423.6-1995）
**检测**: 在 technical_params/qualification_requirements 中检测标准号与实际标准库匹配
**新增规则**: `forbidden_pattern: GB\d{4,}[.\-\d]*[年代号]` + 标准有效性验证

#### P3: 特殊经营资质缺失
**来源**: 宁夏2025-15号（盲人按摩医院射线装置）
**描述**: 销售/使用射线装置需取得辐射安全许可证，但招标文件未将此设为资格条件
**检测**: 检测是否涉及射线装置→检查辐射安全许可证要求
**新增规则**: conditional 类型规则，当技术参数包含"X射线""骨密度仪"等关键词时，强制资格要求包含辐射安全许可证

#### P4: 异常低价未启动审查程序
**来源**: 宁夏2026-6号（沙尘监测项目）
**描述**: 投标报价低于预算45%（93万/205万），评审委员会未按规定启动异常低价审查程序
**检测**: 检测投标报价与预算比例，<45%需启动异常低价说明程序
**新增规则**: numeric_range 规则 + conditional

#### P5: 评分总分与参数扣分不匹配
**来源**: 青海生态环境监测项目
**描述**: 参数满分45分，但△项119项×扣3分=357分，○项20项×1分=20分，扣分总额377分远超45分
**检测**: 计算评分标准中参数扣分上限是否与参数分值匹配
**新增规则**: 语义检测类型

#### P6: 核心产品唯一化
**来源**: 陕西杨凌职业技术学院项目
**描述**: 将"高分辨三维组织扫描成像系统"设为唯一核心产品，变相导致品牌趋同
**检测**: 检测是否仅设置一个核心产品，且该产品参数具有高度指向性
**新增规则**: pattern_required + 语义检测

#### P7: 企业内部非标准型号/代码
**来源**: 陕西铜川陈炉镇项目
**描述**: 使用企业内部非公开代码型号作为参数要求，排斥其他品牌
**检测**: 检测参数中是否包含非标准型号编号（如：字母+数字组合且非公开标准）
**新增规则**: forbidden_pattern 增强版

#### P8: 中小企业声明函真实性
**来源**: 宁夏2026-7号、新疆第八师项目
**描述**: 中标供应商提交的声明函中，将大型批发零售企业声明为中小型制造企业
**检测**: 交叉检查声明函中的制造商经营范围与中小企业认定标准
**新增规则**: AI-SME 检测维度增强

#### P9: ★号参数过度设置
**来源**: 宁夏2026-2号（人民医院手术设备）
**描述**: 多个★号关键参数组合形成整体指向性，每个单独看都能满足3家，组合后仅一家满足
**检测**: ★号参数数量 > N 且类别集中时触发组合排他检查
**新增规则**: parameter_bias_rules.json 新增 pattern

#### P10: 售后服务地域限制
**来源**: 陕西神木水质提升项目、宝鸡陈仓环卫车辆项目
**描述**: 要求投标人在采购人所在地设有固定售后服务网点/机构
**检测**: 检测 qualification_requirements 中是否包含地域性售后要求
**新增规则**: forbidden_pattern 新增

#### P11: 特定机构检测报告限定
**来源**: 青海生态环境监测项目（投诉方主张）
**描述**: 要求提供特定机构（环境保护部环境监测仪器质量监督检测中心）检测报告
**检测**: 检测 technical_params 中是否限定特定检测机构
**新增规则**: forbidden_pattern 新增
