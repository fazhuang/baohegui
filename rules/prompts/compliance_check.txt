## 模板识别指令（重要）
招标文件内容可能带有以下区域标记：
- `<<TEMPLATE>>`...`<</TEMPLATE>>` — 标准范本固定文字，由模板系统预设，**不需要审查**。这些内容在所有同类招标文件中都相同，不属于代理机构自行填写的内容。
- `<<REVIEW>>`...`<</REVIEW>>` — 代理机构实际填写的**变量内容**，**请重点审查这些区域**。

审查规则：
1. 仅从 `<<REVIEW>>` 标记区域中查找违规证据（evidence_text）
2. 不得将 `<<TEMPLATE>>` 区域的文字作为违规证据引用
3. 如果某条敏感表述出现在 `<<TEMPLATE>>` 区域但 `<<REVIEW>>` 区域中无对应问题，视为模板固定用语，不构成违规
4. 如果没有明显的区域标记，则对所有文本进行审查（正常模式）

## 审查要求
请逐条评估每条规则是否满足。对于不满足的规则，请输出：
1. 违反的规则ID — **必须从上方规则列表中选取**，不得编造不存在的规则ID
2. 风险等级（low/medium/high/critical）
3. 具体违反原因
4. 修改建议
5. **证据引用（evidence_text）**：从招标项目数据中引用具体原文，说明违规的依据
6. **法规依据（basis）**：引用具体法律名称和条款号（如《政府采购法实施条例》第二十条），**不得编造不存在的法规或条款**
7. **本条置信度（confidence 0~1）**
8. **是否需要人工复核（needs_human_review）**：有明确原文证据时为 false，纯语义推断时为 true

基于甘肃省政府采购网 558 个投诉案例 + 陕青宁新四省 35+ 真实投诉案例（2025-2026），请重点关注以下违规模式：

| 模式 | 严重度 | 说明 |
|------|--------|------|
| 品牌锁定系列 | 严重 | 要求不同设备/产品为同一品牌 |
| 厂家授权锁 | 高 | 要求投标前取得厂家授权/售后服务承诺函 |

| 售后地域限制 | 中 | 要求采购地设有固定售后网点 |

## 重要约束
1. **rule_id 只能使用上方规则列表中的 ID（如 R001、R002...）或本表列出的 AI_xxx ID**。不得编造其他任何格式的 ID（如 B101、C201、D301 等均为无效）。
2. **不得编造法规依据**。basis 必须是真实存在的法律法规名称和条款号。
3. **evidence_text 必须从项目数据原文中引用**。如果找不到原文证据，将 needs_human_review 设为 true。
4. **没有原文证据支持的问题**，请在 description 开头注明"疑似："，且 needs_human_review 必须为 true。
5. 每个隐含维度最多输出 1 条发现，精要不冗余。
6. 置信度 confidence 设定：有明确原文匹配规则 ≥ 0.9；有原文需语义推断 0.7-0.85；纯语义推理无直接原文 ≤ 0.65。
